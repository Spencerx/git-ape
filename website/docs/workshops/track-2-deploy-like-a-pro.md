# Track 2: Deploy Like a Pro

> **60 minutes** | Intermediate | Azure sandbox required

Deploy multi-resource architectures with security gates, cost estimation, and drift detection.

## What You Learn

- How OIDC, RBAC, and GitHub environments work
- How to deploy multi-resource stacks (Web App + SQL + Key Vault)
- How the security gate blocks insecure deployments
- How to estimate costs and review architecture quality
- How to detect and reconcile configuration drift

## Labs

| Lab | Duration | Description |
|-----|----------|-------------|
| Lab 1: Onboarding | 10 min | Set up OIDC, RBAC, and GitHub environments |
| Lab 2: Web App + SQL | 15 min | Deploy a web app with SQL database and Key Vault |
| Lab 3: Security Deep Dive | 10 min | Break security, see the gate block, fix and re-deploy |
| Lab 4: Cost & Architecture | 10 min | Run cost estimation and WAF architecture review |
| Lab 5: Drift Detection | 5 min | Introduce drift, detect it, and reconcile |

## Prerequisites

- GitHub account with Copilot access
- Azure subscription with Contributor role (Owner for Lab 1)
