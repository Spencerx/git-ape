# Track 1: Zero to Deploy

> **30 minutes** | Beginner | No Azure required

Deploy Azure infrastructure using natural language. No cloud experience needed.

## What You Learn

- How to use `@git-ape` in Copilot Chat
- What a security gate is and why it matters
- How to read a cost estimate and architecture diagram
- Where deployment artifacts are stored

## Labs

| Lab | Duration | Description |
|-----|----------|-------------|
| Lab 1: Setup | 5 min | Set up your environment and verify tools |
| Lab 2: First Deploy | 15 min | Deploy a Function App with one sentence |
| Lab 3: Explore Results | 5 min | Read the architecture diagram, security report, and cost estimate |

## Prerequisites

- GitHub account with Copilot access
- Web browser or VS Code (Codespaces, Dev Containers, or local)
