# Track 3: Platform Engineering

> 110 minutes | Advanced | Azure sandbox required

Master the full CI/CD lifecycle: PR-based deployments, headless mode, multi-environment promotion, policy compliance, and teardown.

## Labs

| Lab | Duration | Description |
|-----|----------|-------------|
| [Lab 1: CI/CD Pipeline](lab-01-cicd-pipeline.md) | 20 min | PR-based plan, review, deploy workflow |
| [Lab 2: Headless Mode](lab-02-headless-mode.md) | 15 min | Issue to auto-generated PR to deployment |
| [Lab 3: Multi-Environment](lab-03-multi-environment.md) | 15 min | Dev/staging/prod promotion strategy |
| [Lab 4: Policy Compliance](lab-04-policy-compliance.md) | 15 min | Azure Policy assessment with CIS/NIST frameworks |
| [Lab 5: IaC Export](lab-05-iac-export.md) | 10 min | Export existing Azure resources to IaC |
| [Lab 6: Destroy Lifecycle](lab-06-destroy-lifecycle.md) | 5 min | Full teardown via PR-based workflow |
| [Lab 7: Advanced Drift Operations](lab-07-advanced-drift.md) | 20 min | Multi-type drift triage, selective reconciliation, scheduled detection |

## Prerequisites

- GitHub account with Copilot access
- Azure subscription with Contributor + User Access Administrator roles
- Completed Track 2 or equivalent Git-Ape experience

See [shared prerequisites](../shared/prerequisites.md) for details.
