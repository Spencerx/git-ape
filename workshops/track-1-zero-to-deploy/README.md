# Track 1: Zero to Deploy

> 30 minutes | Beginner | No Azure required

Deploy Azure infrastructure using natural language. No cloud experience needed.

## Labs

| Lab | Duration | Description |
|-----|----------|-------------|
| [Lab 1: Setup](lab-01-setup.md) | 5 min | Set up your environment and verify tools |
| [Lab 2: First Deploy](lab-02-first-deploy.md) | 15 min | Deploy a Function App with one sentence |
| [Lab 3: Explore Results](lab-03-explore-results.md) | 5 min | Read the architecture diagram, security report, and cost estimate |

## Prerequisites

- GitHub account with Copilot access
- Web browser
