# Track 2: Deploy Like a Pro

> 60 minutes | Intermediate | Azure sandbox required

Deploy multi-resource architectures with security gates, cost estimation, and drift detection.

## Labs

| Lab | Duration | Description |
|-----|----------|-------------|
| [Lab 1: Onboarding](lab-01-onboarding.md) | 10 min | Set up OIDC, RBAC, and GitHub environments |
| [Lab 2: Web App + SQL](lab-02-web-app-sql.md) | 15 min | Deploy a web app with SQL database and Key Vault |
| [Lab 3: Security Deep Dive](lab-03-security-deep-dive.md) | 10 min | Break security, see the gate block, fix and re-deploy |
| [Lab 4: Cost & Architecture](lab-04-cost-architecture.md) | 10 min | Run cost estimation and WAF architecture review |
| [Lab 5: Drift Detection](lab-05-drift-detection.md) | 5 min | Introduce drift, detect it, and reconcile |

## Prerequisites

- GitHub account with Copilot access
- Azure subscription with Contributor role (Owner for Lab 1)

See [shared prerequisites](../shared/prerequisites.md) for details.
